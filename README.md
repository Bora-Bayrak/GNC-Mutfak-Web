# 🍽️ GNC Mutfak Kurumsal Web Sitesi

Bu proje, yerel bir ev yemekleri restoranı için tasarlayıp kodladığım kurumsal web sitesi arayüzüdür. Müşterilerin menüyü kolayca inceleyebileceği, restoranla iletişime geçebileceği ve deneyimlerini paylaşabileceği kullanıcı dostu (UI/UX) bir yapı kurgulamayı hedefledim.

## 🚀 Özellikler
* Kategori bazlı yemek menüsü sergileme (Çorbalar, Ana Yemekler, Tatlılar vb.)
* 5 yıldızlı müşteri yorum ve değerlendirme sistemi arayüzü
* Harita entegrasyonlu ve mesaj gönderimli İletişim/Konum sayfası
* "Bugün Ne Pişti?" şeklindeki günlük menü bilgilendirme alanı
* Modern, sıcak ve restorana uygun renk paleti/tasarım

## 🛠️ Kullanılan Teknolojiler
* HTML5
* CSS3
* JavaScript
